HTML is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and last major HTML version that is a World Wide Web Consortium recommendation . The current specification is knows as the HTML Living Standard .It is maintained by the Web Hypertext Application Technology Working Group, a consortium of the major browser vendors.
